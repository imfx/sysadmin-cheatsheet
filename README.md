### File Management

#### .tar.gz
* `tar -czvf package.tar.gz /your/folder/`
* `tar -xzvf package.tar.gz`

#### .tar
* `tar -cvf packate.tar /your/folder/`
* `tar -xvf package.tar`

#### .gz
* `gzip -9 index.php` where `-9` equals to `--best` for better compression. Opposite is `-1` or `--faster`.
* `gzip -d index.php.gz`

#### .zip
* `zip file.zip folder`
* `unzip file.zip`
